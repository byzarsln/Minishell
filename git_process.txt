

## Branch Açma İşlemleri

- git branch
* bu komut hangi branch da olduğunu gösterir

- git branch <user>
* bu komut bu <user> kullanıcısına ait branch oluşturur 

- git checkout <user>
* bu komut sayesinde branch değiştirirsin

## Push işlemleri

İlk olarak;
- git branch 
işlemi çalıştırarak hangi branch ta oldpunu kontrol et sonrasında;
- git add .
- git commit -m ""
- git push -u origin <user>
* bu komut sayesinde sadece kendi branch ına gönderim yapmış olursun
* sonrasında githubdan repona girerek merge işlemlerini yapabilirsin.

## yapılan değişikliğin çekilmesi işlemi
- git pull origin main
